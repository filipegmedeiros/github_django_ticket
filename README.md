 ## INTRODUÇÃO:

~~~ sh
    É um site para abrir Issues no github. O proposito é servir
    como um sistema de ticket no github, usando uma página simples
    de HTML
~~~
 
| **Github da pessoa que fez 99% do projeto:**            | 
| • [Marlysson](https://github.com/Marlysson)             |





## Como usar ? 

 
 Baixe o Projeto:
~~~ sh
    git clone https://github.com/ticketcomunica/ticket
    cd ticket
    python3 -m venv myenv
     . myenv/bin/activate
    pip install -r requirements.txt
    python manage.py makemigrations
    python manage.py migrate
    python manage.py runserver
~~~

Modifique para sua conta:
~~~ sh
     em settings.py de github_ticket
     Altere:
     USERNAME = 'COLOQUE_SEU_USUARIO_AQUI'
     PASSWORD = 'COLOQUE_SUA_SENHA_AQUI'

     OWNER = 'COLOQUE_O_NOME_DO_DONO_DO_REPO'
     REPOSITORY = 'COLOQUE_O_NOME_DO_REPO'
~~~
