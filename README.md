## Como usar ? 

 
 Usando o git:
~~~ sh
    git clone https://github.com/ticketcomunica/ticket
    cd ticket
    python3 venv myenv
    cd myenv . bin/activate
    cd ..
    pip install -r requirements.txt
    python manage.py makemigrations
    python manage.py migrate
    python manage.py runserver
~~~
